# ABN Platform - AgribizNexus ERP

Frappe/ERPNext v13 SaaS platform for African farmers.

## Features

- Public landing page & farmer registration
- Subscription plans (Basic/Premium/Business)
- Full ERPNext ERP backend
- Multi-tenant support

## Deploy

Connect this repo to [Railway.app](https://railway.app) - it auto-detects docker-compose.yml

## Local Development

```bash
docker compose up -d
```

- Public: http://localhost:8080
- Admin: http://localhost:8080/app
- Login: Administrator / admin
